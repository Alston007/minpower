Installation¶

Basic full installation:

  1. ActivePython 2.7
  2. pypm install minpower
  3. GLPK (Win, Mac)

1. Get Python¶

For new-to-Python folks, install ActivePython 2.7. It makes the installation of a bunch of scientific packages very easy - the way it should be.

For experienced Python users, pip install minpower should work just fine. If that command doesn’t work, it is probably a failure to build scipy/numpy from source - try downloading the pre-built binary for your system from `SciPy <http://www.scipy.org/Download>_.

2. Get Minpower¶

Simply run:

pypm install minpower

3. Get a Solver¶

Minpower has been tested with GLPK, Gurboi, and CPLEX.
