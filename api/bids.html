bidding.parsePolynomial(s)[source]¶

Parse a string into a numpy polynomial object. Heavily adapted from http://bit.ly/polynomialParse.

Can handle simple polynomials (addition and subtraction):

>>> parsePolynomial('7x^2 + 6x - 5')
poly1d([ 7,  6, -5])

or with the explicit * multiplier:

>>> parsePolynomial('7*P^2 + 6*P - 5')
poly1d([ 7,  6, -5])

or even with the terms in some random order:

>>> parsePolynomial('6*P - 5 + 7*P^2')
poly1d([ 7,  6, -5])

Parse a string into a set of multipliers. Heavily adapted from http://bit.ly/polynomialParse.

Can handle simple polynomials (addition and subtraction):

>>> parse_polynomial('7x^2 + 6x - 5')
[-5, 6, 7]

or with the explicit * multiplier:

>>> parse_polynomial('7*P^2 + 6*P - 5')
[-5, 6, 7]

or even with the terms in some random order:

>>> parse_polynomial('6*P - 5 + 7*P^2')
[-5, 6, 7]
