Power Systems Components¶

Defines models for power systems components, including PowerSystem, Bus, Generator, Load, and Line. Each of these objects inherits an optimization framework from OptimizationObject.

Bases: minpower.optimization.OptimizationProblem

Power systems object which is the container for all other components.

Parameters:
  • generators – list of Generator objects
  • loads – list of Load objects
  • lines – list of Line objects
  • num_breakpoints – number of break points to use in linearization of a generator’s bid (or cost) polynomials (equal to number of segments + 1)

  • cost_load_shedding – price of load shedding [$/MWh]
  • dispatch_decommit_allowed – flag - if generators can be decommitted during dispatch
