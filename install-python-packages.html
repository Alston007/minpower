Python packages are open source add-ons that provide great functionality not included in the Python core. They can be installed from the command line by using a package manager like pip (or pypm if you are using ActivePython).

For example:

pip install matplotlib

Note

On Mac/Linux systems you will need to run these commands as root (sudo pip ...)

You can get all the minpower-required packages by running:

pip install pulp coopr numpy matplotlib networkx minpower

An individual package can also be installed by downloading it from the Python package index and then running:

cd download_directory
python setup.py install

You can also download the most recent source of minpower from Github and install that as above.

Package Use in minpower
pulp optimization (current)
coopr optimization (future)
numpy matrix and polynomial math
matplotlib visualization
networkx visualization of graphs (for OPF)
dateutil parses dates and times

Note

If you are using Mac OSX and are having trouble with installing matplotlib through pip - this is a common issue. Check out the matplotlib installation instructions for your options.
